# Comandos

```bash
$ git init

$ git status

$ git add <files>

$ git commit -m 'mensagem'

$ git push <remote> <branch>

$ git pull <remote> <branch>

$ git remote set-url <name> <link>
```

# Branch

```bash
$ git checkout <nome_do_branch>

$ git checkout -b <nome_do_branch>

$ git checkout -d <nome_do_branch>

$ git merge <branch>

$ git fecth --all
```
